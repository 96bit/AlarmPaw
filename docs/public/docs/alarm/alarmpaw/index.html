发送推送#

  1. 打开APP，复制测试URL
  1. 修改内容，请求这个URL

复制推送内容#

收到推送时下拉推送（或在通知中心左滑查看推送）有一个复制按钮，点击即可复制推送内容。

携带参数 automaticallyCopy=1， 收到推送时，推送内容会自动复制到粘贴板（如发现不能自动复制，可尝试重启一下手机）

携带copy参数， 则上面两种复制操作，将只复制copy参数的值

其他参数#

  • isArchive
  • group
  • icon (仅 iOS15 或以上支持）
  • image
  • Timed Notifications
